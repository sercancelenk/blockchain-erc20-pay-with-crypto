## Decentralized Pay With Crypto Implementation ##

### Tech Stack ###
- Node.js
- React
- Solidity Smart Contracts

### Run ###
```
> cd contracts
> truffle migrate --reset
> docker compose up -d
> cd ../ecommerce-dapp-backend
> cd node server.js
> cd ../ecommerce-dapp-frontend
> npm start
> open http://localhost:3000
```
